Troubleshooting

BeebEm will work on Windows XP or later. If you are using XP and do not have DirectX version 9 or later installed then BeebEm will default to using DirectDraw or GDI for rendering. On Vista and Windows 7 DirectX9 should be pre-installed and BeebEm should use this for rendering.

If you have Windows 98 or ME then you can run BeebEm v4.03 which is available on the BeebEm website. You will need to install DirectX9 and IE6. For Windows 98 you will need to install an older release of DirectX 9.0c, such as the Aug 2006 release.

When using DirectX9 in full screen mode the menu bar is not visible. The menus still work but may not be visible! Try moving the mouse cursor to the top left of the screen and press Alt + F to open the File menu. If the menu is not seen then you need to switch out of full screen mode (Alt + Enter) to use the menus.

If you run into other issues drop me an email for help: beebem.support@googlemail.com
