Music 5000 Synthesiser

Hybrid Technology's Music 5000 system is a 16 channel synthesiser. The system can be programmed using the AMPLE language.

The Music 5000 system disk (from Colin Fraser's website) is included in the BeebEm installation. To try it do the following:

  • Enable Music 5000 emulation on the Sound menu
  • Switch to Master 128 emulation using menu option Hardware → BBC Model → BBC Master 128
  • Run the M5000-4.ssd disc using File → Run Disc
  • If loading stops, press Shift + F12
  • When the Studio 5000 Main Menu appears press F9, which will load the jukebox app
  • Use down arrow to scroll down to 'from Notes' and press enter to play

Press F9 again to load the menu again.

To get the Music 5000 system running in Model B mode the AMPLE ROM needs to be configured using Hardware -> Edit ROM Configuration. Load the AMPLE.ROM into an empty slot and press Ctrl-F12. Load the M5000-4.ssd system disc and start the system using '*EXEC RUNNER'.

There are many Music 5000 discs available on the Internet. The music files on many of them can be played using the Music 5000 system disc - boot the system disc first, then load the music disc and press F9. If the jukebox app is not on the disc the files can often be loaded and played using the "file"LOAD and RUN commands in AMPLE (use *CAT to see the files on the disc). For example, the Pachelbel file on the system disc can be played using: "PPACH"LOAD RUN

To play Music 500 files, enable Music 5000 emulation and switch to Master 128 emulation as above and run the "Music500.ssd" disc using File → Run Disc. At the AMPLE % prompt files can be loaded and played with "file"LOAD and RUN. Music 500 files give a "! Too big" error if loaded in Music 5000.
