Acorn Teletext Adapter

BeebEm emulates the Acorn ANE01 Teletext Adapter. It can be used in two ways; Either by loading files containing captured teletext signals, or by connecting to packet servers providing real-time teletext signals.

Teletext capture files, named txt0.dat, txt1.dat, txt2.dat, and txt3.dat, are read from your discimg directory when Capture files is selected in Hardware->Teletext data. You can download some sample teletext data from Jon Welch's site - teletext data.

Real-time teletext packet input is supported using a simple tcp stream interface. An example packet server written in python is available on github. By default BeebEm will attempt four connections to ports 19761 to 19764 on the host PC for the four tuner presets respectively when selecting Hardware->Teletext data->IP: localhost.

Alternative addresses and ports set via the Preferences.cfg file can be used by selecting Hardware->Teletext data->IP: Custom destination (this is only intended for use over high bandwidth local network connections).

The ATS-3.0-1.rom must be added to the ROM configuration and Hardware->Teletext Adapter On/Off enabled.
