State Images

BeebEm can save the entire state of the emulated BBC Micro to a state file. The state file can then be loaded back in later to restore the BBC Micro back to the saved state. State files are put in the 'BeebState' folder by default.

Saving and load states is useful for saving your position in a game or saving a BASIC program without using disc images. There are various menu options and keyboard shortcuts for saving and loading state files.
