Welcome to BeebEm!

BeebEm is a BBC Micro and Master Series emulator. It enables you to run BBC Micro software on your PC. BeebEm should work on most PC systems running Windows XP or later.

See the Getting Started section for information on how to run and use BeebEm.

BeebEm is distributed under the terms of the GNU General Public License, as described in the COPYRIGHT.txt file. Source code is available from GitHub so you can compile and modify BeebEm yourself.

If you have any problems, questions or suggestions for improvements please raise an issue using the BeebEm issue tracker on GitHub.
