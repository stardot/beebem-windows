Uninstalling BeebEm

If you wish to uninstall BeebEm then select the Uninstall option in the BeebEm folder in the Windows Start Menu or use the Add or Remove Programs option in the Windows Control Panel/Settings. Note that any files you have added to your My Documents\BeebEm folder will not be removed.
