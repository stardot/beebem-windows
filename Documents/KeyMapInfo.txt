Key map data
------------

There is one line per virtual key code on the PC keyboard.  Each line
contains Beeb key info (row, col, shift_state) for an unshifted key press
and a shifted key press.

Special values for row, col are:

-2      break key
-3, -3  sound tuning key -
-3, -4  sound tuning key +
-9      ignore key press
